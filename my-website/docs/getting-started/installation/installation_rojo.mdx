# Installation

To install Shard via Visual Studio and Plugin, follow these steps:

1. Open Visual Studio and create a new project or open an existing one.
2. In the Solution Explorer, right-click on the `ReplicatedStorage` folder and select `Add` > `ModuleScript`.
3. Name the module script `Shard`.
4. Copy and paste the Shard code into the `Shard` module script.
5. Save your project and run it to see Shard in action.
