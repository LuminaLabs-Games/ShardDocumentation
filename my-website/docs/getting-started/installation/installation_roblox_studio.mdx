# Installation

Install shard using a model file

1. Open Roblox Studio and create a new project or open an existing one.
2. In the Explorer panel, right-click on the `ReplicatedStorage` folder and select `Insert Object`.
3. Choose `ModuleScript` from the list and name it `Shard`.
4. Copy and paste the Shard code into the `Shard` module script.
5. Save your project and run it to see Shard in action.
